# Skills Repository

This repository contains local Codex design skills and plugins prepared for reuse and distribution.

## Included

- `design-spec-plugin`
  - A SaaS design spec plugin for stable admin-console and enterprise product UI generation.

## Notes

- The plugin is packaged as a local Codex plugin.
- Update plugin metadata in `design-spec-plugin/.codex-plugin/plugin.json` before public release if you want to replace the placeholder website URLs with real ones.
