# SaaS Page Brief Template

Use this template when describing a new page. It is intentionally short so it can be pasted into a prompt with minimal friction.

```md
Page type:
Primary goal:
Target users:
Language mode:
Primary result surface:
Need KPI cards:
Key modules:
Important data or metrics:
Primary actions:
Secondary actions:
States needed:
  - default
  - loading
  - empty
  - error
Responsive priority:
Special constraints:
```

## Example

```md
Page type: search table page
Primary goal: manage customer support tickets
Target users: support managers and agents
Language mode: en-US
Primary result surface: table
Need KPI cards: yes, keep within 4
Key modules: filter bar, KPI summary, ticket table, row drawer, bulk actions
Important data or metrics: open tickets, overdue tickets, SLA risk, resolution rate
Primary actions: assign, change priority, close ticket
Secondary actions: export, save view, open detail
States needed:
  - default
  - loading
  - empty
  - error
Responsive priority: desktop first, tablet acceptable
Special constraints: dense but easy to scan, must keep status colors clear
```
